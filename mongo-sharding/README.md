# pymongo-api

## Как запустить

Запускаем mongodb с настроенным шардингом и репликми и приложение

```shell
docker compose up -d
```

Заполняем mongodb данными

```shell
./scripts/mongo-sharding-init.sh
```

В логах работы скрипта будет выведена информация о добавленных данных и их распределении по шардам

## Как проверить

Для получения информации о созданном кластере откройте в браузере http://localhost:8080
